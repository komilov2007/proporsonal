# charos-proposal

Premium romantic proposal website for Charos.

## Tech
- React 19
- Vite
- TypeScript
- Framer Motion
- canvas-confetti
- CSS-only ring box animation

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel
- Build command: `npm run build`
- Output directory: `dist`
